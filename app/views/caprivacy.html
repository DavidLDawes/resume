Privacy Policy for Koch Zap

The Koch Zap smart phone application for Android does not collect personal data from its Users. The Koch Zap application relies on the Android smart phone and Google's Android frameworks for many of it's features and Koch Zap has no direct control over those applications. Google Analytics is used to provide usage statistics.

Types of Data collected

Google Analytics will track which views are being accessed and for how long. No personal data is provided to us, just totals across all users.
No personal data is collected.
No information about scan usage is collected.

The use of the collected Data

We will use the analytic informtion - amount of installs, frequency of use, duration of use, and which views are used - to help us plan what, if anything, we wish to do to improve the Koch Zap application.

Definitions and legal references

Personal Data

Any information regarding a natural person, a legal person, an institution or an association, which is, or can be, identified, even indirectly, by reference to any other information, including a personal identification number.
This application does not keep any personal data of any sort.

Usage Data

Google Analytics will track usage: time spent, views visited, installs and uninstalls.
Actual steps taken on those views and the associated data (scan results, lookup and search results, user input values) are not tracked.

User

The individual using this Application.
